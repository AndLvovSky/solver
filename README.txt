Загальні зауваження:
- щоб запусти програму відкрийте файл dist/Solver.jar
- для відкриття необхідно, щоб була встановлена Java-машина версії не менше Java SE 8
- код класів знаходиться в папці src/solver
- результати тестувань алгоритму знаходяться в папці test

----------------------
Користування програмою
----------------------
Основні кроки:
1. Перенесіть деякі підходи у список використаних
2. Перенесіть деякі задачі у список використаних
3. Нажміть кнопку вирішити
Інші можливості:
- додати свою задачу
- відмінити використання деяких задач чи підходів
- змінити обмеження часу
- відображати неуспішні вирішення задач
- у довідці можна подивитись детальну інформацію про кожен підхід
- не забудьте підказки, що з'являються при наведенні на компоненти
